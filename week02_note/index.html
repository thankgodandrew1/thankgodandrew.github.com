Class Basic Syntax

what is a class? in OOP, a class is an extensible program-code-template for creating objects, providing initial values for state (member variables) and implementations of behavior (member functions or methods). The class statement initiates a JavaScript class. Properties and methods are assigned in the constructor() method. The constructor() method is called each time a class object is initialized, the basic class syntax is in this format:

class MyClass{
property = value; // property

constructor(...) { // constructor
// ...
}
method(...) {} // method

get method(...) {} // getter method
set method(...) {} // setter method
...

[Symbol.iterator]() {} // method with computed name (symbol here)
// ...}

In the above code it's cool to know that MyClass is also a function, you could get the proof of that if you declare a class and check for the type using the typeof operator keyword. The above code is a clear representation of how a basic class syntax should look like, methods, getters and setters are found or written to MyClass.prototype.

Object methods, "this"

Based on the reading it's common that an object method needs to access the information stored in the object to do it's job. This is were one of the benefits of this arises. with the "this" keyword a method can access an object with it's properties. In an example format the value of "this" is the object before the dot, it is the one used to call the method.

let person = {
name: "Samuel"
age: 19
}

Based on the explanation above the "this" keyword can be used in an object method to access the person object properties. How is this done, it is done using the "this" keyword, this.name accesses the name property which is "Samuel" the value of this will be the person object. this can also be used in any function, even if it's not a method of an object.

Export and Import

Export and import directives have several syntax variants. Based on further personal study i discovered that through the help of ES6, we can create modules in JavaScript. In a module, there can be classes, objects, functions, and so on. To make all these available in another file we use or can use export and import these keywords are used for exporting and importing one or more member in a module. There are different syntax we can use to both export, and import.
