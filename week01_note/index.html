Syllabus

It is expected that students understands the lessons taught about JavaScript, HTML, and CSS from the previous course (WDD 230).

Some softwares and resources are required for students to carry out task, Git and GitHub is required and also a good IDE such as Visual Studio Code is required. The course Structure is said to be unusual, students are meant to spend the first 9 weeks in teams working on a project, while from the 10th to 14th week students work with their team on a final project of their choice. There are lots of outcomes from the course all which will develop confidence in students as a Frontend developer.

Managing the Modern Frontend Workflow

Before now managing the modern frontend workflow appears to be difficult, but developers have developed tools to help simplify project management, based on the readings there are some tools we can use to manage the mordern workflow, they fall into three categories:

  • Package managers: Package manager keeps track of what software is installed in our computer, and allows us to easily install new softwares, upgrade or delete existing softwares. We will be using npm as our package manager
  • Bundlers: Bundler is an assemblage tool we will use to combine, transpile, concatenate, and minify two or more packages. We are using Snowpack as our bundler
  • Task Managers: These keeps track of what needs to be done and when, we wil also be using npm

Course Setup

Setting up the course needs some core requirements, which includes installing a good code editor, the recommended one is Visual Studio Code, also install the Node Package Manger, Git and GitHub, Microsoft Teams, Join a team (the total team size should be 3-4), and lastly one needs to create a Trello account.
